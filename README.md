# Encerrador de Ciclos

## Pré-requisitos

- Docker
- GNU Make

## Comandos básicos

Iniciar a aplicação:

```bash
make up
```

Parar a aplicação:

```bash
make down
```

Iniciar a aplicação forçando a construção das imagens:

```bash
make build
```

Log do bot:

```bash
make log.bot
```

Acessar o container do bot:

```bash
make shell.bot
```
